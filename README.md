# Gestão do Conhecimento LBR - Frontend Next.js

Projeto de frontend para o sistema de Gestão do Conhecimento da LBR Engenharia.

## Tecnologias
- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Prisma (PostgreSQL)
- NextAuth.js

## Deploy
O deploy é feito automaticamente na Railway ao fazer push para a branch main.
